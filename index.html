Front-End Tools

ReactJS Release: 2013 React is declarative, component-based JavaScript library for building user interfaces.
AngularJS Release: 2010 AngularJS (commonly reffered to as AngularJS 1.X) is a JavaScript framework for building SPAs with an MVC architecture.
VueJS Release: 2014 Vue.js (pronounced /vjuː/, like view) is an open-source JavaScript framework for building user interfaces.
Elm Release: 2016 Elm is a domain-specific programming language for declaratively creating web browser-based graphical user interfaces.
Polymer Release: 2015 Polymer is an open-source JavaScript library for building web applications using Web Components.

Back-End Tools

NodeJS Release: 2009 Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. Uses an event-driven, non-blocking I/O model taht makes it lightweight and efficient.
ASP.NET Core Release: 2016 ASP.NET Core is the next generation of ASP.NET. It is a modular framework that runs on both the full .NET Framework, on Windows, and the cross-platform .NET Core.
